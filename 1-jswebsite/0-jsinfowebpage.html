Loops

Loops allow us to run a block or section of code continuously until a condition is met. There are 3 main types of Loop statements, which are explained briefly below.

forLoops

forLoops need to contain 3 expressions. The first one declares a variable, the second states a condition that must be met before iteration of the loop, and the third states and expression to be evaluated after the loop has run each time.

forOfLoops

Only works with iterable values. This allows it to work well with arrays, but cannot work with objects.

forInLoops

Used for iterating values in an object that are normally non iterable.
